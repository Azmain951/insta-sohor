Questions and Answers

1. Session storage stores data temporarily. On the other hand, Local storage stores data permanently.

2. Stored data in session storage get deleted as soon as the browser/tab is closed. The local storage stored data remains intact even after closing browser/tab.

3. Local storage stored data can be removed using JavaScript code. Session storage stored data gets removed automatically when the session ends.

Event loop is a JavaScript concept which works runtime of JS code. JS code can contains many events and other instructions. Some are synchronous and some asynchronous.Event loop handle these things at runtime. It runs a loop for the events where the synchronous instructions get executed one by one from the stack. Then, It executes the queued instructions in the FIFO(first in first out) order. Event loop handles all these things on runtime.

JavaScript runs on V8 engine on browser. The V8 engine parse the JS source code and makes an Abstract syntax tree. Then it interpret the code and make the runtime process more faster using JIT(just in time) compilation technique. Then it provides a optimized machine code and then converted into Bytecode.The V8 engine uses turboFan, Ignition for JIT compilation, code optimization and garbage collection.
